Class EncodeToolsSerializer

An implementation of Serializer that internally uses @znetstar/encode-tools for encoding/decoding. Any serialization methods supported by @znetstar/encode-tools (today JSON, MessagePack, CBOR, and BSON) are supported when using this Serializer.
