constructor

  • new Server(transports?: Transport | Array<Transport>, methodHost?: any): Server

  • Overrides EventEmitter2.__constructor

    • Defined in Server.ts:202

    Creates a RPC server using one or more transports. Using multiple transports the server can listen for RPC requests on multiple protocols.

    example

    let tcpTransport = new TCPTransport(1234); let methods = { foo: (bar) => { console.log(bar); } }; new Server(tcpTransport, methods);

    Parameters

    • Optional transports: Transport | Array<Transport>

      An array of transports or single transport that the server will listen on.

    • Default value methodHost: any = {}

      An object containing methods that will be executed upon request from the client.

    Returns Server

Protected batch

  • batch(messages: Array<Request | Notification>, clientRequest: ClientRequest): Promise<void>
    • Defined in Server.ts:280

    This method is called when the server recieves a batch request (an array with multiple requests). Requests will be executed sequentially and responses will be in the same order as the corresponding requests. Notifications will be executed but of course will generate no response. If a request has an error the error will stand in place of a response, but subsequent requests will still be executed.

    listens

    Transport#batch

    Parameters

    • messages: Array<Request | Notification>

      An array of either Request or Notification objects.

    • clientRequest: ClientRequest

      The ClientRequest object that contains information on the request that was made (e.g. client's ID).

    Returns Promise<void>
